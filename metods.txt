Backtrader — это фреймворк для написания торговых стратегий на Python.
В Backtrader есть несколько предопределенных методов, которые вы можете переопределить в своей стратегии.
Вот основные из них с описанием их назначения:


__init__():
Этот метод вызывается при инициализации стратегии.
Здесь вы можете установить параметры, создать индикаторы и подготавливаться к началу торговли.
В этом методе вы можете настроить все, что вам нужно, перед началом выполнения стратегии.

next():
Этот метод вызывается на каждом шаге (или свече) данных, которые вы загружаете в стратегию.
Здесь размещается основная логика вашей торговой стратегии.
Все ваши торговые сигналы и сделки обычно запускаются именно в этом методе.

nextstart():
Этот метод вызывается один раз перед началом первого вызова метода next().
Это полезно, если вам нужно выполнить какую-либо логику или инициализацию,
которая должна произойти только один раз перед началом обработки данных.

stop():
Этот метод вызывается в конце выполнения стратегии.
Здесь вы можете добавить код, который должен выполняться после завершения торговли,
например, для записи результатов или отчета о производительности.

notify_order(order):
Этот метод вызывается при изменении состояния заказа. Он позволяет вам обрабатывать различные состояния заказа
(например, открытие, исполнение, отмена и т. д.) и реагировать на них соответствующим образом.

notify_trade(trade):
Этот метод вызывается при изменении состояния торговли.
Он позволяет вам отслеживать результаты сделок, например, прибыли и убытки, а также может использоваться для записи статистики.

notify_cash():
Вызывается при изменении наличных средств на счете.
Это может быть полезно для отслеживания ликвидности и остатка на счете.

notify_data():
Этот метод позволяет получать уведомления о статусе загружаемых данных
(например, если данные загружены, инициализированы и т. д.).

nextgoal():
Этот метод может быть полезен для обработки событий, связанных с изменением целей или ограничений в стратегии,
однако он используется не так часто, как остальные методы.
